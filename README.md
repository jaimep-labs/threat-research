# Threat Research

This repository focuses on the analysis of threat actor behavior, techniques, and trends using publicly available intelligence.

## Scope
- Threat actor profiling
- TTP analysis (MITRE ATT&CK)
- Campaign and malware family research
- Detection-oriented analysis

## Content
- Threat reports
- Technique breakdowns
- Mapping to MITRE ATT&CK
- Defensive insights

## Sources
- Public threat intelligence reports
- Vendor blogs and whitepapers
- Open-source intelligence (OSINT)
